C++知识

类

struct(结构)是成员默认为public的class(类)

class的成员默认是private

对象的复制

私有成员

构造函数、explicit、初始化列表

构造函数

隐式转换和explicit

建议：一般来说，单参数的构造函数都声明为explicit，除非有特别的原因。

当然，explicit也可以用于无参或者多个参数的构造函数中。

构造函数初始化列表

inline、const、mutable、this、static

inline

在类定义中实现成员函数inline，会被当做内联函数来处理。

内联函数只是对编译器的建议，能不能inline成功，依旧取决于编译器。

const

mutable 不稳定的、容易改变的

mutable的引入也是为了突破const的限制

末尾有const修饰的成员函数是不允许修改成员变量值的，但是使用mutable修饰的成员变量在常量成员函数中是允许修改的。

返回自身对象的引用——this

this用在成员函数中是一个隐藏起来的函数参数，表示的是指向本对象的指针。

***this表示该指针指向的对象也是本对象 **

this本身是一个指针常量，总是指向对象本身，不可以让this再指向其他地方

static成员

static成员变量(静态成员变量): 不属于某个对象，而是属于整个类

这种成员变量可以通过对象名来修改(也可以通过对类名来修改)

一旦通过该对象修改了这个成员变量的值，则其他该类对象也能够直接看到修改后的结果

类名::成员变量名 的方式可以对这种成员变量进行引用

成员函数也可以用static修饰，代表该函数不属于某个对象，而是属于类

类内初始化、默认构造函数、 =default; 和 =delete;

类内初始化

const成员变量的初始化

对于类的const成员，只能使用初始化列表来初始化，而不能在构造函数内部进行赋值操作

默认构造函数

没有参数的构造函数称为默认构造函数

一旦程序员书写了自己的构造函数，那么在创建对象的时候，

必须提供与书写的构造函数形参相符合的实参，才能成功创建对象。

=default；和 =delete；

=default;
=delete;

让程序员 显示地禁用某个函数

拷贝构造函数

①单参数的构造函数，一般声明为explicit，以防止出现代码模糊不清的问题

②拷贝构造函数，一般都不声明为explicit

小总结

一些会调用拷贝构造函数的情形
